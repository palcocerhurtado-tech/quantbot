# SEM Playbook - Archon Consultancies

## Goal

Capture high-intent demand for:

- `consultoria de ia`
- `agencia de ia`

with dedicated landing pages and direct-response messaging.

## Budget and match logic

- Start with exact and phrase match only.
- Split campaigns by intent, not by broad topic.
- Send each keyword group to its matching landing page.
- Exclude low-intent educational searches in the first phase.

## Conversion setup still needed

To launch real SEM campaigns, you still need:

- Google Ads account access
- Microsoft Ads account access if you also want Bing
- Conversion tracking ID
- Search Console and Google Analytics connected
